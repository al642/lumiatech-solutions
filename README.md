# LumiaTech Solutions

A deployment-ready React + Vite marketing site for a tech solutions company.

## Structure

- `index.html` app entry
- `public/` static assets
- `src/components/` shared UI blocks
- `src/pages/` homepage sections
- `src/styles.css` global styling

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
